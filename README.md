### 🛠️ Featured Projects

#### [🤖 AgileAI-Orchestrator](https://github.com/sainathkhot/AgileAI-Orchestrator)
Multi-agent orchestration platform automating Agile software development.
- **5 agents** — PM, Architect, Scrum, Reviewer, Sprint Planner — built as a real **LangGraph `StateGraph`** (nodes, edges, checkpointer)
- Human-in-the-loop review node + persistent checkpointing
- Local LLMs via Ollama (DeepSeek, Qwen, Llama 3) + RAG + FastAPI
